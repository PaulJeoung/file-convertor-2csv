## python 사용울 위한 가상 환경

- 가상환경 추가 및 실행

    1. python3 -m venv venv
    2. source venv/bin/activate
    3. 가상환경 종료
        - deactivate

## 파일 실행

*.py 실행을 위한 module 설치 (pip)

이후 각 파일들을 실행

1. /job-reasearch
    - pip install selenium webdriver-manager
    - pip install beautifulsoup4

2. /cloudwatch-file-convert
    - cloudwatch에 있는 데이터를 spreadsheet 파일로 변환
    - pip install pandas
